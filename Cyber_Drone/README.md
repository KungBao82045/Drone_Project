# About this project
Control your drone using voice commands! With the implementation of AI, your drone will communicate smarter. It will follow commands by human response and act as a hostile robotic drone.
As for 09/04/25, this is my latest project, to demonstrate my creativity. 

# Note
Because you need to connect your drone through the WiFi, as well as connecting to the internet in order to get the API endpoint, you need a WiFi adapter. With WiFi adapter, you can connect your laptop to the internet, and wifi adapter to your drone.
